# Deploy qo'llanmasi

## Yangilanishlarni GitHub ga yuklash

Har qanday o'zgartirishlardan keyin:

```bash
# Windows (PowerShell)
.\deploy.ps1

# Linux/Mac
./deploy.sh
```

## Render.com avtomatik deploy

Render.com GitHub ga ulangan bo'lsa, har bir push'dan keyin avtomatik deploy bo'ladi.

## Qo'lda deploy

```bash
git add .
git commit -m "Yangilanish: [tavsif]"
git push
```
